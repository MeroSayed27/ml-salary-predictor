 
# ML Salary Predictor

This is a simple Machine Learning project that predicts salary based on years of experience.

## Tools Used
- Python
- Pandas
- Scikit-learn

## What it does
- Loads dataset
- Trains Linear Regression model
- Predicts salary based on experience

## Example
Input: 5 years experience  
Output: predicted salary value

## Goal
Practice ML workflow for job preparation and portfolio building.
